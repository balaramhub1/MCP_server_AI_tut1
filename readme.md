Required Libraries:\
- requests - A simple, yet elegant HTTP library for Python.\
- pypdf - A library for reading and writing PDF files.\
- sentence-transformers - A library for computing dense vector embeddings representations for sentences and paragraphs.\
- mcp - A library to create/ use the MCP servers.


Install the dependencies using pip:
 - pip install -r requirements.txt

Command to run the server:
 - uv run main.py
